## ex01. QGIS 설치

### 공식 홈페이지에서 다운로드
- 다운로드 살펴보기: https://qgis.org/

### 프로그램 및 기능 확인
```
appwiz.cpl
```
- 프로그램 추가/제거에서 확인

### winget을 이용한 설치
```powershell
winget list qgis
winget search qgis
winget install -e --id OSGeo.QGIS_LTR
```

특정 버전 설치:
```powershell
winget show OSGeo.QGIS --versions
winget install -e --id OSGeo.QGIS --version <버전번호>
```

PowerShell 업데이트(참고):
```powershell
winget search Microsoft.PowerShell
winget upgrade Microsoft.PowerShell
winget install -e --id Microsoft.PowerShell
```

## ex02. 공간 데이터 다루기 (p73)

4. **행정경계 불러오기**
   - 폴리곤 → 라인 → 포인트 변환
5. **낚시터 실습**

## ex03. 위치정보가 있는 속성 데이터 다루기 (p77)

1. 서울열린데이터광장 > 검색: `공중화장실` > 내려받기(CSV)
2. **데이터 확인 및 정제**
   - 파일명, 필드명, 폴더명 등 영어로 변경
   - 불필요한 항목 삭제
3. QGIS > 레이어 > 레이어 추가 > **구분자로 분리된 텍스트 레이어를 추가**
   - CSV 불러오기
4. 옵션 설정
   - 인코딩 설정
   - X: `LON`, Y: `LAT`
   - 좌표체계: `EPSG:4326`

## ex04. 위치정보가 없는 속성 데이터 다루기 (p84)

   - 대상 필드: `SGG_NM`

## ex05. 도형 표현도 제작 (p93)

1. **결합(조인)**: `admin_sgg.shp` + `인구총조사.csv`
2. 속성 테이블 > 필드 계산기
   - 추가 필드: `내국인_비율` = `내국인/총인구*100`
   - 추가 필드: `외국인_비율` = `외국인/총인구*100`
3. `admin_sgg` > 속성 > **도표**
   - 유형: 파이차트
   - 추가 값: `내국인_비율`, `외국인_비율`
4. 크기 설정
   - 조정된 크기 > 속성: `총인구` > 최대: 찾기
5. `admin_sgg` > 속성 > **라벨링**
   - 도표 유형: 텍스트 도표 / 히스토그램

## ex06. 단계 구분도 제작 (p97)

## ex08. 디지타이징: 한강 (p108)

1. 참조용 배경지도 불러오기: TMS for Korea 등
2. 레이어 > 레이어 생성 > **새 shp 레이어**
   - 이름: `han_river.shp`
   - 도형 유형: 폴리곤
   - 좌표계: 배경지도에 맞게 설정
3. 새 필드 추가
   - 이름: `name`, 유형: 텍스트 > 필드 목록 추가 > 확인
4. 디지타이징
   - 폴리곤 객체 추가 > 한강 디지타이징 > 마우스 우클릭(폴리곤 폐합)
5. 속성 정보 입력 > 편집 모드 해제 > 저장
6. 속성 테이블 열기 > 필드 계산기
   - 새 필드: `area`, 유형: 실수, 표현식: `$area` > 확인

## ex09. 스내핑 설정 디지타이징

1. 설정 > 옵션 > **디지타이즈 작업**
2. 스냅 설정
   - 스냅 활성화
   - 기본 스냅 모드: 꼭짓점
   - 기본 스냅 허용 오차: `10`
   - 꼭짓점 편집용 탐색 반경: `10` > 확인
3. 스냅 툴바 > 스냅 활성화 > 편집 모드 시작
4. 스냅 활성화 상태로 디지타이징 > 편집 모드 종료 > 저장
5. 객체 편집
   - 노드 추가
   - 클릭 & 드래그: 노드 이동
   - 클릭 + Delete: 노드 삭제
6. 도구 모음 > **고급 디지타이징 툴바** > 편집 시작 > 객체 단순화

## ex10. 버퍼
**목표: 8차선 이상 도로부터 500m 이상 떨어진 소방서 찾기**

1. `road_link2.shp`, `firestation.shp` 불러오기
2. `road_link2` 레이어 선택 > 선택 툴바 > **표현식으로 피처 선택**
   - 조건: `"LANES" >= 8`
   - 피처 선택 결과: 931 / 10,506 > 닫기
3. 벡터 > 지오프로세싱 > **버퍼**
   - 선택한 피처만 체크
   - 거리: `500`
   - 결과물 융해 체크 > 실행
4. 벡터 > 조사 도구 > **위치로 선택**
   - 대상: `firestation`
   - 관계: 분리됨(disjoint)
   - 비교 레이어: 버퍼 산출물 > 실행
5. 프로젝트 > 새로 생성 > 무시(저장하지 않음)

## ex11. 인터섹트(교차영역) vs 클립
**목표: 한강이 통과하는 행정구역 찾기**

1. `admin_sgg.shp`, `river.shp` 추가
2. 벡터 > 공간 처리 도구 > **교차영역(형상 교차)**
   - 입력 레이어: `admin_sgg.shp`
   - 교차 레이어: `river.shp`
3. 벡터 > 공간 처리 도구 > **클립(형상 유지)**

> 인터섹트는 속성값이 그대로 유지되며 교차 영역의 형상을 반환하고, 클립은 입력 레이어의 형상만 대상 경계로 잘라낸다는 점에서 차이가 있습니다.

## ex12. 디졸브
**목표: 11개 유형 토지이용도를 3개 유형으로 재분류**

1. `landuse.shp`, `gangnam_dong.shp` 불러오기
2. `landuse` 레이어를 `gangnam_dong` 모양으로 **클립(자르기)**
3. 공간 처리 툴박스 > 벡터 도형 > **도형 수정**
   - 수리 방법: Linework > 실행
4. 편집 시작 > 필드 추가: `재분류`
5. 선택 툴박스 > 표현식으로 선택
   - `이용범례 = '주택지'` > 필드 계산기(기존 필드 업데이트) > `재분류` = `'주거지'`
   - `이용범례 = '상업 및 업무시설지'` > `재분류` = `'상업지'`
   - `!'주택지' and !'상업 및 업무시설지'` > `재분류` = `'녹지 및 기타'`
6. 벡터 > 공간 처리 도구 > **디졸브(융해)**
   - 디졸브 필드: `재분류`
7. 레이어 > 속성 > 심볼 설정
8. 프로젝트 > 새로 생성 > 무시

## ex13. 복합: 강남구 초등학교 반경 200m 이내 청소년불건전시설 찾기

1. `gangnam_dong.shp`, `gangnam_elesch.shp`, `gangnam_POI.shp` 불러오기
2. **금지구역 설정**: 초등학교 반경 200m 영역 찾기
   - 벡터 > 공간 처리 도구 > 버퍼 > `초등학교` > `200`m > 실행
3. **금지구역 내 POI 추출**
   - 벡터 > 공간 처리 도구 > 잘라내기(클립) > `POI` × `200m 버퍼` > 실행
4. **청소년불건전시설 찾기**
   - 표현식을 이용한 객체 선택
   - 조건: `"NAME" LIKE '%당구%' or "NAME" LIKE '%PC%'`
   - 피처 선택 결과: 10건

## ex14. 밀도지도(KDE, heatmap) 작성
**목표: 서울시 노인교통사고 다발지역 분석**

1. `admin_sgg.shp`, `senior_att.shp` 불러오기
2. 공간 처리 툴박스 > **열지도(Heatmap)**
   - 포인트 레이어: `senior_att.shp`
   - 탐색 반경: `1,000`
   - 픽셀 크기: `100`
   - Weight from field: `sago_cnt`(사고건수)
   - Kernel shape: `Quartic`
3. 레이어 속성 > 심볼
   - 단일 밴드 유사색상 > 색상 램프 반전
   - 투명도: `75%`
4. 스타일링: 시도 경계, 사고 지점, 도로망도 등 함께 표시

## ex15. 공간보간: 서울시 연평균 기온 지도 만들기

1. `YAVRG_TMPRT_2008.shp`, `admin_sgg.shp` 불러오기
2. 공간 처리 툴박스 > 보간법 > **역거리 가중(IDW) 보간법**
   - 벡터 레이어: `YAVRG_TMPRT_2008`
   - 보간 속성값(Attribute): `MEAN_08` > 추가
   - Search Range: `global`
   - Search Radius: `1,000`
   - 범위: `admin_sgg` 레이어 범위로 설정
   - 픽셀 크기: `100` > 실행
3. 래스터 > 추출 > **마스크 레이어로 래스터 자르기**
   - 입력 레이어: 보간 산출물
   - 마스크 레이어: `admin_sgg`
   - 잘라낸 래스터의 범위를 마스크 레이어 범위와 일치시킴
   - 인풋 래스터 해상도 유지 > 실행
4. **지도 스타일링**
   - 레이어 속성 > 심볼 > 단일 밴드 유사색상
   - 그레이디언트 색상표 반전 > 적용
